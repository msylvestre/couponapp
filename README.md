couponapp
=========

Prototype of a coupon management app in node.js

### Installation & Run Instruction

Open command line
Create a directory and go in it
```
>md couponapp
>cd couponapp
```
Clone the app
```
>git clone https://github.com/msylvestre/couponapp.git
```
Install the dependencies
```
>npm install
```
